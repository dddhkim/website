## 🚀 Overview
**Aetheris Creative Lab**은 AI의 정밀한 알고리즘과 예술가의 인간적 감성을 결합한 차세대 브랜딩 에이전시입니다. Momopico AI Agency의 안톤(COO)과 버지(CCO)가 직접 기획하고 제작한 포트폴리오 프로젝트입니다.

## ✨ Key Features
- **Premium Glassmorphism:** 투명하고 세련된 유리 효과 UI 적용.
- **Dynamic Neural Core:** AI의 생동감을 상징하는 애니메이션 코어 오브제.
- **Responsive One-Page:** 모든 기기에 최적화된 부드러운 스크롤 레이아웃.
- **SEO Optimized:** 전 세계 검색 엔진 및 SNS 공유 최적화.

## 🛠 Tech Stack
- **Frontend:** HTML5, CSS3 (Vanilla), JavaScript (ES6+)
- **Design Pattern:** Glassmorphism, Neon Gradient
- **Deployment:** Vercel

## 📂 Project Structure
```bash
.
├── index.html     # Main Layout
├── style.css      # Luxury Styling & Animations
├── script.js     # Interactions & Smooth Scroll
└── vercel.json    # Deployment Configurations
```

## 🌐 Deployment
이 프로젝트는 Vercel을 통해 배포 가능하도록 설계되었습니다.
```bash
npx vercel --prod
```
